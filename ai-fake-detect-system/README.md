# AI Fake Detect System

AI Fake Detect System 是一个面向图片、文本、语音、视频等内容的 AI 伪造内容识别与辅助取证展示系统。

## 技术栈
- Frontend: ShipAny-based Web UI
- Backend: Python Flask
- Core Features: Image and Text Detection
- Extended Features: Audio and Video Analysis Display

## Directory Structure
- frontend/: 前端系统
- backend/: 分析服务
- docs/: 项目文档
- cases/: 案例样本
- outputs/: 分析输出结果
